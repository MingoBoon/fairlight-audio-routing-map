5.1 Mix Guide

Mix in 5.1 with a map, not a maze.

An interactive signal-flow map, step-by-step build guide and plain-English glossary for creating a professional 5.1 film & TV mixing template in Resolve's Fairlight page. Click anything. Learn everything.

Every box tells you what it is.
Every line tells you how to build it.

Built for documentary/broadcast delivery — and for people who've never routed a bus in their life.

🔍

Trace any signal

Click a track, bus, stem or VCA and see everything that feeds it, everything it feeds, and the full downstream chain — lit up on the map.

🖱️

Click a wire

Every connection line explains what that route does and the exact Resolve 21 clicks to create it — Bus Outputs, Bus Sends, Bus Assign, VCA Assign.

💬

Plain-English mode

Each element opens with a jargon-free explanation ("a reverb bus is a shared echo machine…") before the technical detail.

📖

Glossary everywhere

Hover any dotted-underlined term — LUFS, dBTP, dialog-gated, M&E — for an instant definition. Plus a full filterable glossary.

✅

Build checklist

A 12-step interactive checklist with saved progress, including the full 21-bus format table (5.1 / Stereo / Mono / 5.0 — exactly which and why).

📊
